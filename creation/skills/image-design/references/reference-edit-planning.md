# Reference And Edit Planning

Use this reference when the task starts from existing images.

## Reference Image Roles

- Style anchor: color, texture, lighting, medium, mood.
- Character anchor: face, hair, clothing, proportions, distinctive features.
- Product anchor: shape, material, package, logo placement, scale.
- Composition anchor: layout, camera angle, object positions.
- Element source: specific objects to fuse into a new design.

For each reference, state what should be preserved and what may change.

## Edit Types

- Inpainting: remove or replace objects in a selected area.
- Outpainting: extend canvas beyond current borders.
- Background removal/replacement: separate subject and create or place a new background.
- Upscaling/restoration: improve resolution, faces, noise, or damage.
- Style transfer: change visual language while preserving subject.
- Relighting: change light direction, warmth, mood, or shadows.
- Local post-processing: crop, resize, convert, compress, strip metadata, watermark.

## Non-Destructive Rules

- Preserve the original file.
- Work on a copy.
- Name output paths before writing.
- For multi-step edits, keep intermediate outputs when useful.
- Review every semantic edit before continuing to the next step.

## Risk Notes

- Exact face, logo, product, and text consistency is not guaranteed.
- Multi-reference workflows depend on provider limits.
- Private or commercial reference images require explicit consent before upload.
