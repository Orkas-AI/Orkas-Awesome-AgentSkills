# Video Ad Storyboard

## 基础结构

短视频广告要先抓注意力，再建立信任，最后降低行动阻力。

1. 0-3 秒：停留理由。
2. 3-8 秒：痛点或场景。
3. 8-18 秒：产品解决方式。
4. 18-25 秒：证明镜头或反对意见处理。
5. 最后 3-5 秒：CTA。

## 平台节奏

| 平台 | 节奏 | 内容重点 |
|---|---|---|
| TikTok / Reels | 快，前 3 秒极重要 | UGC、反差、场景、评论感 |
| YouTube Shorts | 信息密度可稍高 | 演示、教程、对比 |
| Meta Feed | 静音观看多 | 字幕、产品清晰、利益点 |
| 抖音 | 内容场景和购买理由 | 痛点、演示、直播承接 |
| 小红书 | 种草和真实体验 | 使用感、生活方式、避坑 |
| Amazon video | 产品演示和可信度 | 功能、尺寸、使用步骤 |

## Hook 类型

- 痛点直击。
- 场景代入。
- 结果展示，避免夸大前后对比。
- 反常识。
- 评论回应。
- 开箱 / 演示。

## 证明镜头

| Claim | 需要的镜头 |
|---|---|
| 易清洗 | 水冲 / 拆卸 / 对比清洁时间 |
| 便携 | 放包、车载、单手拿 |
| 容量大 | 实物参照、装入数量 |
| 稳固 | 摇晃、承重、结构特写 |
| 静音 | 使用环境和声音对比，避免绝对化 |
| 省空间 | 使用前后收纳对比 |
| 防漏 | 倒置 / 摇晃，但要符合真实测试 |

## A/B 测试优先级

1. Hook。
2. 首屏画面。
3. 证明镜头。
4. CTA。
5. 时长。
6. 口播风格。

## 指标解释

| 指标 | 含义 | 可能问题 |
|---|---|---|
| 3 秒观看率 | 开头是否抓人 | Hook 或首屏弱 |
| 完播率 | 内容是否连贯 | 信息过多、节奏慢 |
| CTR | 是否有点击动力 | CTA 或利益点弱 |
| CVR | 商品页是否承接 | 页面、价格、信任不足 |
| 评论问题 | 用户疑虑 | FAQ 和详情页要补 |
| 退货率 | 预期是否准确 | 视频夸大或说明不足 |
