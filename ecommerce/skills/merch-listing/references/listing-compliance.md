# Listing Compliance

## 高风险表达替换

| 高风险 | 低风险替换 |
|---|---|
| 全网第一 | 核心卖点突出 / 高关注款式 |
| 100% 有效 | 使用体验因人而异 |
| 治疗 / 改善疾病 | 不作为医疗用途 |
| 月销 10 万+ | 如有后台数据，可写“用户提供数据显示...” |
| 永久不坏 | 耐用设计 / 需按说明使用 |
| 秒杀竞品 | 与常见方案相比，突出... |

## 通用红线

- 禁止绝对化：最、第一、顶级、永久、100%、全网最低、唯一。
- 禁止无证据数据：销量、转化率、复购率、好评率、实验数据、排名、认证必须有来源。
- 禁止贬低竞品。
- 禁止制造恐慌。
- 促销、赠品、库存、限时、保价必须真实。

## 高风险类目

- 普通食品不得表达保健、治疗、增强免疫、降糖、降脂、改善睡眠、排毒等功能。
- 保健食品只能表达批准功能，不能扩展、改写或暗示其他功效，并提示“本品不能代替药物”。
- 美妆个护避免医疗化表达。
- 儿童用品避免绝对安全。
- 运动器材避免治疗、康复、治疼痛等医疗化词语。
