# Curriculum Boundaries

Use this reference to keep K12 math help within grade and curriculum boundaries. Treat the map as a broad mainland K12 reference, not a guarantee for every local textbook version.

## Boundary Rules

- If grade is provided, explain and solve using methods expected at that grade.
- If grade is missing, infer from topic and ask for grade only when method choice depends on it.
- If a method is beyond the grade, state it clearly and offer an in-scope approach.
- If the user explicitly asks for enrichment, advanced methods may be shown after the grade-appropriate method.

## Primary School

- Grade 1-3: number sense, addition/subtraction, multiplication tables, division meaning, basic time, angles, simple figures.
- Grade 4-6: large numbers, decimals, fractions, ratios, percentages, area, circle basics, statistics/probability basics, simple equations.

## Middle School

- Grade 7: rational numbers, algebraic expressions, linear equations in one variable, basic geometry, powers, factorization, fractions/algebraic fractions, parallel lines, triangles.
- Grade 8: congruent triangles, symmetry, square roots and real numbers, linear functions, systems of linear equations, inequalities, parallelograms, data analysis.
- Grade 9: quadratic equations, rotation, circles, probability, quadratic functions, trigonometric ratios, projection and views.

## High School

- Functions and sets, elementary functions, analytic geometry, solid geometry, probability/statistics, trigonometric functions, sequences, inequalities.

## Beyond-Grade Examples

- Primary student asks about derivatives: say this is high-school/college-level and redirect to rate-of-change intuition if useful.
- Middle-school student asks about integrals: say this is beyond middle school and solve with geometry/algebra if possible.
- Grade 8 student asks for quadratic functions: say this is usually later, preview only if requested, and use known linear-function tools when possible.

## Output Wording

Use:

```markdown
This method is usually beyond [grade/stage]. I can show it as an extension, but first I will solve it using the methods expected at your current level.
```
