# Prompt Style Library

Use this structure for stable visual prompts.

## Prompt Fields

- SUBJECT: main subject and key attributes.
- CONTEXT: scene, use case, target audience, and platform.
- COMPOSITION: camera angle, layout, visual hierarchy, negative space, crop.
- STYLE: medium, era, visual language, texture, rendering approach.
- LIGHTING: mood, color temperature, contrast, shadows, atmosphere.
- TEXT: exact short text, placement, readability requirements.
- TECHNICAL: aspect ratio, resolution intent, image type, output purpose.
- CONSTRAINTS: avoid list, brand rules, safety, realism, references to preserve.

## Common Use Cases

- Social cover: strong focal subject, clear title area, platform-safe crop, high contrast.
- Product image: product geometry, material, reflections, clean background, commercial lighting.
- Poster: headline space, layered hierarchy, theme symbol, controlled palette.
- Avatar: centered subject, simple background, readable silhouette at small size.
- Infographic: minimal generated text, clear icon areas, final text added in layout tools.
- Series: fixed palette, fixed framing, consistent character/product/style anchors.

## Style Guidance

- Prefer concrete visual language over vague adjectives.
- Use "in the style of" cautiously; avoid imitating living artists or protected brands when commercial use is implied.
- For Chinese text, keep generated text short and plan post-production for longer copy.
- For brand work, describe brand constraints instead of asking the model to recreate protected marks.
