# Citation Style

Use APA 7 unless the user requests another citation style.

## In-text citations

Use author-date citations:

```markdown
Retrieval practice has been associated with durable learning benefits across multiple classroom contexts (Smith, 2021; Lee & Kumar, 2023).
```

For three or more authors, use `et al.` in text:

```markdown
Several reviews identify spacing and retrieval as complementary learning mechanisms (Garcia et al., 2022).
```

When a page number is needed for a direct quote, include it:

```markdown
The authors define the intervention as "..." (Nguyen, 2020, p. 14).
```

Avoid direct quotes unless the wording itself matters.

## Reference list

Include as much bibliographic detail as is available:

```markdown
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. Journal or Publisher, volume(issue), page range. https://doi.org/...
```

For web sources:

```markdown
Organization Name. (Year, Month Day). Title of page. Site Name. URL
```

If no date is available:

```markdown
Organization Name. (n.d.). Title of page. Site Name. URL
```

## Requirements

- Every in-text citation must appear in the reference list.
- Every reference-list item should be cited in the report.
- Do not invent missing authors, dates, titles, page numbers, DOIs, or URLs.
- If citation metadata is incomplete, state what is missing.
- Alphabetize references by the first author's surname or organization name.

## Citation density

Use enough citations to make the evidence trail auditable. Important claims usually need one or more citations in the same paragraph. Major conclusions should be supported by multiple independent sources when available.

## Source transparency

When citing non-peer-reviewed, commercial, or advocacy sources, identify the source type in prose where it affects interpretation.
