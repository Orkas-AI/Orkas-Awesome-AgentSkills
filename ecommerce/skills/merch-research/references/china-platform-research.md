# China Platform Research

## 选品先判断 6 件事

1. **需求是否真实**：看搜索词、内容热度、评论量、复购场景、季节性，警惕短期热点伪需求。
2. **竞争是否可进入**：看头部店铺数量、评价壁垒、价格带、同质化程度。
3. **利润是否撑得住**：扣除平台费、运费、包装、退货、客服、广告和损耗。
4. **供应链是否稳定**：关注起订量、交期、质检、备货压力和售后率。
5. **内容是否好表达**：抖音/小红书适合视觉强、场景强、对比强、演示强的产品。
6. **合规和售后是否可控**：食品、保健、美妆、儿童、数码电器、医疗器械要谨慎。

## 品类类型

| 类型 | 特征 | 适合策略 |
|---|---|---|
| 标品 | 参数明确、价格透明、同质化高 | 拼价格、服务、供应链、评价壁垒 |
| 非标品 | 款式、风格、审美影响大 | 拼内容、主图、差异化、上新速度 |
| 复购品 | 消耗快、周期稳定 | 拼留存、套餐、私域、评价 |
| 冲动品 | 低客单、强场景、内容驱动 | 拼短视频、达人、首图冲击 |
| 高客单品 | 决策长、信任要求高 | 拼证据、参数、售后、品牌背书 |
| 季节品 | 旺季明显 | 拼备货节奏、提前投放、库存控制 |

## 平台差异

| 平台 | 机会 | 风险 |
|---|---|---|
| 淘宝 / 天猫 | 搜索和店铺承接强，适合长线经营 | 关键词竞争、评价壁垒 |
| 京东 | 适合标品、品牌、服务型商品 | 用户对价格和服务敏感 |
| 拼多多 | 低价、规格、供应链优势明显 | 价格战、毛利压缩 |
| 抖音 / 快手 | 内容驱动、演示型商品机会大 | 素材消耗快、退货和冲动消费 |
| 小红书 | 种草、审美、生活方式强 | 转化链路长，真实性要求高 |
| 1688 | 供应链和价格参考 | 不能直接等同零售需求 |

## 字段采集建议

- 基础字段：平台、商品名、链接、价格、规格、销量/付款人数、评价数、店铺、发货地、采集时间。
- 视觉字段：首图卖点、主图数量、是否有视频、场景图、对比图、参数图、证据图。
- 运营字段：优惠、包邮、退换、发货时效、客服承诺、赠品、活动标签。
- 风险字段：差评主题、退货风险、尺寸争议、破损风险、功效风险、资质风险。

## 利润测算

毛利不能只算“售价 - 进货价”。

- 单件毛利 = 售价 - 进货价 - 平台费 - 运费 - 包装 - 退款损耗 - 广告成本 - 售后成本。
- 毛利率 = 单件毛利 / 售价。
- 盈亏平衡销量 = 固定成本 / 单件毛利。
