# Root Cause And Actions

## 差评归因

1. **产品可改**
   - 质量、结构、功能、材质、配件。
   - 反馈研发、供应链或质检。

2. **页面可改**
   - 尺寸没讲清、效果承诺过高、图片误导。
   - 改标题、主图、详情页、FAQ。

3. **运营可改**
   - 发货慢、赠品漏发、客服口径不一致。
   - 改流程和客服 SOP。

4. **预期管理**
   - 用户买前理解偏差。
   - 强化使用边界、适用人群、不适用场景。

5. **外部不可控**
   - 快递、平台规则、极端个例。
   - 记录但不要过度归因。

## 从评论到行动

| 评论信号 | 对应动作 |
|---|---|
| 反复问尺寸 | 主图加尺寸图，详情页加规格表 |
| 说不会用 | 增加使用步骤图和短视频 |
| 说效果不明显 | 降低承诺，增加适用边界 |
| 说包装破 | 改包装，首屏增加发货保护说明 |
| 说客服回复慢 | 改客服 FAQ 和响应流程 |
| 说和图片不符 | 重拍主图，减少过度修饰 |
| 说贵 | 增加材质、服务、套装价值解释 |

## 优先级

按四个维度排序：

- 影响范围：影响评论占比或销售阻力。
- 可执行性：是否能快速改页面、客服、包装、质检或商品。
- 成本：改动所需人力、库存、供应链和广告成本。
- 紧急程度：是否涉及安全、合规、批次质量或高频退款。
