# Product Visual Prompts

## 商品视觉目标

1. 用户一眼看懂卖什么。
2. 产品主体不能变形。
3. 关键材质和规格要可信。
4. 场景服务卖点。
5. 平台比例和文字安全区适配。
6. 商用授权和版权风险可控。

## Hero Frame 方向

| 方向 | 适合商品 | 视觉重点 |
|---|---|---|
| 极简棚拍 | 高端、数码、美妆、家居 | 光影、材质、留白 |
| 成分飞散 | 食品、饮料、护肤 | 新鲜、原料、风味 |
| 使用场景 | 家居、户外、宠物、母婴 | 人群、环境、使用价值 |
| 对比结构 | 工具、清洁、功能型商品 | 前后、旧新、参数 |
| 礼品氛围 | 节日、套装、高客单 | 包装、仪式感 |
| UGC 真实感 | 抖音、小红书、TikTok | 手持、生活化、轻修饰 |
| 科技感 | 数码、电器、智能设备 | 结构线、界面、光效 |

## 图片 Prompt 结构

包括主体、场景、光线、材质、构图、色彩、平台比例、文字处理和约束。

通常先不要让模型生成文字，后期再排版。

## 视频 Motion Prompt 结构

包括初始画面、镜头运动、产品运动、背景运动、节奏、时长、比例和验收标准。

5-10 秒适合产品展示；15 秒以上需要分镜。

## 负面提示词

- distorted product
- changed logo
- misspelled text
- extra labels
- extra hands
- deformed packaging
- unrealistic reflection
- cluttered background
- misleading medical claim
- competitor brand
- unauthorized celebrity

中文约束：

- 不改变商品形状。
- 不生成虚假认证。
- 不出现竞品品牌。
- 不添加未经确认的文字。
- 不出现医疗、治疗、功效暗示。
- 不使用未授权人物肖像。

## 验收标准

图片：

- 产品形状和主要结构未变。
- 包装文字没有明显错字或乱字。
- 没有未授权人物、竞品、商标。
- 场景与目标人群匹配。
- 画面支持一个明确卖点。

视频：

- 产品没有融化、变形、漂移。
- 镜头运动稳定。
- 时长和比例符合平台。
- 关键卖点在前 3 秒可见。
- 无误导性功效表现。
