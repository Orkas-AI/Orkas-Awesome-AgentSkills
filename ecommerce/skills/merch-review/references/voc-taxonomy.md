# VOC Taxonomy

## 样本偏差

评论不是直接结论。先判断：

- 差评样本更容易被极端体验放大。
- 好评可能受返现、赠品、默认好评影响。
- 近期评论比历史评论更能反映当前批次。
- 追评通常比首评更能反映真实使用。
- 不同 SKU 混在一起会掩盖问题。

## 主题体系

| 一级主题 | 二级主题 |
|---|---|
| 产品质量 | 做工、耐用、异味、损坏、稳定性 |
| 功能体验 | 是否有效、是否易用、学习成本、兼容性 |
| 规格匹配 | 尺寸、容量、颜色、材质、适配范围 |
| 物流包装 | 发货慢、破损、漏发、包装简陋 |
| 客服售后 | 响应慢、退换难、解释不清 |
| 价格价值 | 不值、比别家贵、活动后价差 |
| 预期落差 | 图片不符、描述不清、效果不如预期 |
| 安全合规 | 过敏、儿童安全、食品安全、电器风险 |

## 输出规范

- 每个主题至少给 1-3 条脱敏证据片段。
- 百分比必须说明分母。
- 样本不足时不要给强结论。
- 没有评分字段时，情感结果标为推断。
