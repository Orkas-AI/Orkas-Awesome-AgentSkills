# Visual Copywriting

## 平台视觉差异

| 平台 | 首图重点 | 详情页重点 | 常见误区 |
|---|---|---|---|
| 淘宝 / 天猫 | 搜索点击率、核心卖点、品牌信任 | 痛点递进、证据、规格、FAQ | 文案过满、卖点太散 |
| 京东 | 品牌可信、参数清晰、自营/服务感 | 参数、质保、安装、售后、对比 | 只讲情绪，不讲规格 |
| 拼多多 | 价格价值、规格数量、到手利益 | 低价合理性、规格确认、售后保障 | 只喊便宜，缺信任 |
| 抖音 / 快手 | 内容场景、3 秒吸引、使用前后 | 视频承接、评论疑问、购买理由 | 直接搬搜索电商详情页 |
| Amazon | 主图干净合规、图片顺序讲清功能 | A+、comparison chart、review objection | 图片塞过多文字 |
| Shopify | 品牌感、生活方式、场景信任 | 品牌故事、材质、评价、FAQ | 只有促销感 |

## 主图 5 张逻辑

1. **首图：点击理由**。产品主体 + 核心利益 + 可信证据。
2. **痛点图：用户代入**。典型问题场景 + 低风险表达 + 解决方向。
3. **差异图：为什么买你**。结构/材质/工艺/服务差异 + 证据状态。
4. **场景图：什么时候用**。2-3 个高频场景 + 人群标签 + 使用利益。
5. **成交图：降低下单阻力**。规格、售后、包装、价格、CTA。

## 详情页信任递进

1. 首屏：适合谁、解决什么问题。
2. 痛点：真实场景，不制造恐惧。
3. 方案：产品结构、材质、工艺或服务如何解决。
4. 证据：检测、资质、评价、参数、对比。
5. 场景：不同人群/场合的使用方式。
6. 规格：尺寸、容量、材质、适配范围。
7. FAQ：反对意见处理。
8. 法律声明：高风险类目必须保守。
