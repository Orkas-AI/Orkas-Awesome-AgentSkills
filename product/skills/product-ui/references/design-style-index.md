# Design Style Index

Use this index to choose one UI style reference under `references/design-styles/`. Read only the selected file.

## Recommended Defaults

| Product Type | Suggested References |
|---|---|
| SaaS landing page | `stripe.md`, `vercel.md`, `linear.app.md` |
| Developer tool | `vercel.md`, `cursor.md`, `raycast.md`, `supabase.md` |
| AI product or chat UI | `claude.md`, `mistral.ai.md`, `elevenlabs.md`, `vercel.md` |
| Fintech | `stripe.md`, `revolut.md`, `coinbase.md`, `wise.md` |
| Productivity app | `notion.md`, `linear.app.md`, `superhuman.md` |
| Ecommerce or consumer brand | `airbnb.md`, `apple.md`, `spotify.md`, `pinterest.md` |
| Internal dashboard | `sentry.md`, `posthog.md`, `clickhouse.md`, `ibm.md` |
| Creative/design tool | `figma.md`, `framer.md`, `clay.md`, `webflow.md` |
| Enterprise/B2B | `ibm.md`, `hashicorp.md`, `mongodb.md`, `sanity.md` |

If the user does not specify a style, prefer the app's existing design. If there is no existing design and the user says to choose, default to `vercel.md` for a clean, neutral UI.
