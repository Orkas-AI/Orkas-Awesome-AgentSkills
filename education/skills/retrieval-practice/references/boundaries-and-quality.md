# Boundaries And Quality

Use this checklist before returning retrieval practice materials.

## Quality checklist

Each set should pass these checks:

- The questions require memory reconstruction.
- The answers are not given away by the question wording.
- The item mix matches learner level and time since learning.
- The questions target meaningful knowledge, not trivia.
- Known misconceptions are addressed when provided.
- Answer notes tell the teacher what to accept and what to watch for.
- The implementation script keeps the activity low-stakes.

## Boundary checklist

Do not:

- Create direct-submit homework answers.
- Create high-stakes exam papers or official assessments unless the user provides formal criteria and asks for draft review.
- Claim that the questions are guaranteed to improve scores.
- Collect personal student data.
- Produce questions that shame, rank, or label learners.
- Use retrieval practice as a substitute for teaching missing prerequisite knowledge.

## Distinguish from nearby skills

Use a homework/quiz tutor when:

- A learner wants live one-question-at-a-time practice.
- The user wants their answer checked.
- The user asks for hints on a specific homework problem.

Use a study planner when:

- The user wants a study schedule, spaced review calendar, check-ins, or weekly plan.
- The task is deciding when to practice, not authoring the practice materials.

Use this skill when:

- The user wants a set of active-recall questions, answer notes, and a low-stakes implementation script.

## Factual alignment

If the user supplies lesson notes, textbook excerpts, slides, or a syllabus, align questions to that material.

If no source material is supplied:

- Use general subject knowledge.
- State that the teacher should check terminology against what was taught.
- Avoid overly specific claims that depend on a particular curriculum.
