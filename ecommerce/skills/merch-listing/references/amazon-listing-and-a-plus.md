# Amazon Listing And A+

## 页面目标

Amazon Listing 同时服务：

1. 搜索匹配：关键词覆盖和相关性。
2. 点击决策：主图、价格、评分、coupon、Prime/FBA。
3. 转化决策：图片顺序、bullet、A+、评价、QA。
4. 风险降低：尺寸、材质、兼容性、退换、使用边界。

## 标题方法

- 品牌 / 核心类目词。
- 关键功能或材质。
- 适用场景 / 人群。
- 规格 / 数量 / 尺寸。
- 避免堆砌重复词和不可证实 claim。

## Bullet 五点分工

| Bullet | 作用 |
|---|---|
| 1 | 核心利益：为什么值得看 |
| 2 | 关键功能：如何解决痛点 |
| 3 | 材质 / 规格 / 兼容性：降低误购 |
| 4 | 使用场景：让用户代入 |
| 5 | 售后 / 包装 / 礼品 / 风险降低 |

每条 bullet 建议包含 Feature -> Benefit -> Evidence / Detail。

## 图片顺序

1. 主图：白底、主体清楚、合规。
2. 核心卖点图：最强功能 / 利益。
3. 尺寸规格图：减少误购和退货。
4. 场景图：展示使用人群和环境。
5. 对比图：客观参数，不贬低竞品。
6. 细节图：材质、结构、配件。
7. 包装 / 套装 / 礼品图：降低预期差。
8. 视频：演示使用、安装、效果边界。

## A+ 模块

A+ 不重复 bullet，而是补足：

- 品牌信任。
- 场景故事。
- 材质 / 工艺解释。
- 对比表。
- 使用步骤。
- FAQ。
- 系列产品导购。

移动端优先：每屏只讲一个重点，图片中文字要少。

## 评论反对意见映射

| 评论痛点 | 页面动作 |
|---|---|
| 尺寸不合适 | 增加尺寸图、适配表、测量方法 |
| 材质不如预期 | 增加材质细节图和真实描述 |
| 安装难 | 增加步骤图和视频 |
| 易坏 | 增加结构说明、质保、使用边界 |
| 颜色不符 | 降低过度修图，说明色差 |
| 包装破损 | 强化包装图和售后承诺 |
| 效果不明显 | 降低承诺，加入适用 / 不适用场景 |
